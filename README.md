# customer_behavior_analysis
data analytics project showcasing customer behavior using python sql and power Bi
Customer Behavior Dashboard Hey there! This project is all about understanding customer shopping behavior using data and visual storytelling. I built this dashboard to explore how customers purchase, how much they spend, what they buy, and how different factors like age, gender, and subscription status impact sales.

The goal Turn raw customer data into clear, useful business insights.

What this project does?
This dashboard helps answer questions like:
How many customers do we have overall?
What’s the average purchase amount?
How satisfied are customers based on review ratings?
Do subscribed users generate more revenue?
Which categories and age groups drive the most sales?
All of this is presented in a simple, easy-to-understand dashboard.
